This script will download all the IPTV channel JSON files from the iptv-org
github repo and copy them to a ftp server. A copy is also saved to a
database for easy access.

In addition all the icons that are stored on imgur, which are inaccessible
from the UK, are downloaded to the MythFrontendQML config directory in
~/.mythqml/iptv/icons/ for easy access. You may have to use a VPN to
download the icons if you are in the UK.


Copy secrets_stub.json to secrets.json and update the ftp access credentials
in that file.
